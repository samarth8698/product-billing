# Product Billing System

A React + TypeScript billing application built using Redux Toolkit for state management.

## Live Demo

https://rococo-bunny-d824ac.netlify.app

## GitHub Repository

https://github.com/samarth8698/product-billing

## Tech Stack

- React
- TypeScript
- Redux Toolkit
- Tailwind CSS
- Vitest

## Features

- Product Selection
- Quantity Management
- Billing Calculation
- Special Offers & Discounts
- Savings Calculation
- Final Bill Generation
- Redux State Management
- Unit Testing

## Special Offers

- Apple: Buy 3 Get 1 Free
- Orange: 10% Discount on 2 or more items
- Banana: Buy 5 Pay For 4

## Unit Tests

Implemented using Vitest.

Tested Features:
- Increment Quantity
- Decrement Quantity
- Clear Bill

## Installation

npm install
npm run dev
