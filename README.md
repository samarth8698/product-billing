# Product Billing System

A React + TypeScript billing application built using Redux Toolkit for state management.

## Live Demo

<https://rococo-bunny-d824ac.netlify.app>

## GitHub Repository

<https://github.com/samarth8698/product-billing>

## Tech Stack

- React
- TypeScript
- Redux Toolkit
- Tailwind CSS
- Vitest

## Features

- Product Selection
- Quantity Management
- Basket Management
- Billing Calculation
- Special Offers & Discounts
- Savings Calculation
- Individual Savings Display
- Final Bill Generation
- Redux State Management
- Unit Testing

## Installation

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Project

```bash
npm run build
```

### Run Tests

```bash
npm run test
```
